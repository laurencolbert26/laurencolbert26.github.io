Fantastical End

Fantasical End is a game built for the Girls Who Code Summer Immersion Program 2017. The focus of the game is to acknowledge mental illnesses and the stigma against getting help for those illnesses. The gameplay is a text only RPG, in which the player must fight various monsters that are representive of symptoms of different mental illnesses. The choices that the player makes impact which monsters must be fought.
